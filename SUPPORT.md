# Support — WordBloom

## Reporting Bugs

If you encounter a bug, please [file an issue on GitHub](https://github.com/rohit-wadhwa/Word-Bloom/issues/new?template=bug_report.md).

The more details you provide, the better we can assist you. Please include:
- Steps to reproduce the issue
- Expected vs actual behavior
- Your browser, device, and OS
- The level number and a screenshot if applicable

## Feature Requests

Have an idea for a new feature? [Submit a feature request](https://github.com/rohit-wadhwa/Word-Bloom/issues/new?template=feature_request.md).

## General Feedback

For general comments, questions, or feedback, visit the [GitHub Discussions](https://github.com/rohit-wadhwa/Word-Bloom/discussions) page.
